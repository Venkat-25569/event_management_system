                                   EVENT MANAGEMENT FULLSTACK PROJECT                                            
A full-stack web application designed to manage and organize university events efficiently with role-based access and centralized data handling.

FEATURES:
* Secure Authentication using JWT
* Role-Based Access (Student, HOD, Admin, Coordinator)
* Event Creation & Management
*  Event Registration System
*  Centralized Database
* Role-based Dashboards
* Rubrics for live score board
* Online certificate generation
* Attendance Management

TECH STACK:
*FRONTEND: HTML, CSS, JavaScript
*BACKEND :Java, Spring Boot
*DATA BASE:MySQL


PROJECT STRUCTURE:
event-management-system/
├── backend/
├── frontend/
├── database/
└── README.md
 
 AUTHENTICATION:
* Uses JWT (JSON Web Token) for secure login
* Supports multiple user roles
